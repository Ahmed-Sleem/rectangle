# Other Open-Source PM & Building Blocks

## Taiga
- Agile Scrum/Kanban specialist  
- Clean UX for small teams  
- Weak Gantt/portfolio/cost  
- Stack: Python (Django) historically + Angular/JS frontends  
- Good reference for backlog UX only  

## Leantime
- Lean/startup PM, goals, TODOs  
- PHP stack  
- Lighter alternative  

## Vikunja
- Task/todo self-host  
- Go backend  
- Not full PM  

## Focalboard (Mattermost Boards)
- Trello-like  
- Pairs with Mattermost chat — interesting **chat+boards** combo  
- Not construction  

## Kanboard
- Minimal Kanban PHP  
- Plugin ecosystem  
- Good for internal simple workflows  

## ProjectLibre / GanttProject
- **Desktop** Java CPM/Gantt  
- MS Project alternatives offline  
- No modern multi-tenant SaaS  
- Useful for understanding CPM UI paradigms & file formats  

## OrangeScrum
- Agile + some Gantt  
- Mixed maintenance reputation — verify before basing product  

## Tuleap
- Full ALM (code + trackers + docs)  
- Enterprise open source  
- Heavy  

## phpCollab / Collabtive
- Legacy PHP collab — historical only  

## NocoBase
- Plugin data-model platform  
- “AI employees” direction  
- **Build construction OS as apps on platform** — serious option for speed  

## Budibase / Appsmith
- Internal tools low-code  
- Approvals, admin, dashboards fast  
- Not end-user field product core  

## AppFlowy / Outline
- Docs/wiki only — PM library piece  

## Twenty
- OSS CRM — not PM; sales side  

## WorkLenz etc.
- Newer self-host PM entrants — monitor
