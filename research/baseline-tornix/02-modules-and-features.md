## A. Core product modules (detailed)

### 1. Dashboard / Command Center / لوحة التحكم
**Pitch (AR):** مشروعك بنظرة واحدة، قبل قهوتك الأولى.  
**Pitch (EN idea):** Instant stakeholder snapshot of what’s on track, at risk, and needs a decision today — no hunting, no calls.

Typical “Today’s Focus” widgets shown:
- Risks & issues count + affected projects  
- Task completion % (+ week-over-week)  
- Due this week (with urgent count)  
- Projects count (active subset)  
- Due tasks list with priority  
- Risk analysis breakdown (Critical / High / Medium / Low)  
- AI recommendations with severity + “Take action”  
- Project engagement metrics  

**Ailigent name:** Command Center — every domain in one live executive overview (KPIs, variances, cash flow).

### 2. Projects / المشاريع
One place for one site or fifty. Create from scratch **or let AI build the full structure from a BOQ (Bill of Quantities)**. Track progress, phases, full portfolio view.

Fields shown in demos: status, progress %, health %, owners/members, date range, description.

### 3. Tasks / المهام
Kanban or list; priorities; deadlines; auto follow-up. Sub-checklists (e.g. 2/4). Assignees. “Nothing lost in chaos.”

Statuses in UI demo: To Do · In Progress · Completed.

### 4. Calendar / التقويم
Unified view of deadlines, milestones, meetings, inspections. Day / week / month. Prevents missed pours, handovers, client reviews.

### 5. PM Library / مكتبة إدارة المشاريع
Institutional construction knowledge: searchable, reusable, always improving. Store templates, standard BOQs, lessons learned, compliance checklists. AI finds items in seconds.

Sample knowledge cards:
- Lessons learned & continuous improvement  
- Effective risk methodologies & processes  
- Quality management fundamentals  
- Stakeholder engagement strategies  
- Triple constraint (time / cost / scope)  
- Understanding the project lifecycle  

### 6. Documents / المستندات
Single source of truth for contracts, drawings, reports, approvals. Version control to prevent costly build-from-wrong-revision errors. “Right drawing, approved version — find it in 5 seconds.”

Also: CAD drawings support listed in module icon grid; auto-extract items from blueprints (App Store).

### 7. Chats / المحادثات
Stop losing decisions in WhatsApp groups. Conversations linked to project / task / document. AR + EN. AI summarizes long threads in one click. Threads (Slack-style on mobile). Voice notes, media, share-to-Tornix.

### 8. Mail / البريد
Professional email inside the platform. Auto-link messages to projects and tasks. Context-organized inbox.

### 9. Meetings / الاجتماعات
Schedule/join from app. **Auto-transcription**, smart summaries, share decisions and tasks. Video summary support on mobile. Speaker routing controls.

### 10. Risks & Issues / المخاطر والمشاكل
Log risks before they become disasters. Impact assessment, mitigation plans, **5×5 risk matrix**. AI scores severity/likelihood and suggests priorities. Escalation, duplicate detection (changelog). Critical-path impact.

AI recommendation examples from demo:
- Accelerate steel procurement — supply delay hits critical path by 12 days (Critical)  
- Redistribute Team B — 140% overloaded, burnout risk (High)  
- Schedule QA inspection before phase-3 foundation pour (Medium)  

### 11. Requests & Approvals / الطلبات والموافقات
Smart approval cycle from submission to final sign-off. Status tracking, comments, attachments, “who must act now.”  
**Ailigent:** Approval Flow Builder — design any path as a **visual DAG**. Requests & templates to standardize intake.

### 12. Team / الفرق والأعضاء
Org people from senior PM to subcontractors. Roles, permissions, workload visibility. Know who is overloaded before mistakes. Fine-grained project + org permissions (App Store). Team sentiment analysis. Skills coverage.

### 13. Procurement / المشتريات
AI-assisted request generation, vendor management, document tracking. Create RFQs, RFPs, POs, contracts (Play Store).

### 14. Costs / التكاليف
Baseline vs actual, change orders, invoices, cash flow. **CPI, SPI, EVM live.** Cost analysis module. Standalone Costs section on mobile.

### 15. Strategy / الاستراتيجية
Turn org vision into measurable goals. **Balanced Scorecard**, live KPIs, performance by perspective: Financial, Customer, Process, Learning & Growth. Link board goals → projects → benefits.

### 16. PMO Dashboard / لوحة PMO
Full portfolio: budgets, risks, strategic alignment, program health — charts and interactive tables. Executive / C-level analysis. Portfolio view.

### 17. Time tracking / تتبع الوقت
Log hours per task; exportable reports for billing/analysis (App Store + module grid).

## C. AI layer

### Positioning
- “AI in every feature” / الذكاء الاصطناعي في كل ميزة  
- “Your project answers you. Ask anything. Instantly.”  
- AI assistant persona on site: **المهندس-آدم (Engineer-Adam)** — System agent  
- Chat starter areas: AI insights, starter questions, KPIs, FAQs, team updates, quick-start templates  
- Domains in assistant UI: Automation · Projects · Chats · Agents  

### AI Super Agent / Tornix AI Twin
Autonomous reasoning agent (not a side chatbot) acting within granted authority:

- Self-directs a daily routine across every project (no prompting)  
- Reasons over schedule, cost, risk to predict problems  
- Drafts reports/updates for one-click review  
- Escalates only exceptions that matter  

### Domain agents named on marketing site
- Scheduling agent — وكيل الجدولة  
- Cost agent — وكيل التكلفة  
- Risk agent — وكيل المخاطر  
- Reporting agent — وكيل التقارير  
- Documents agent — وكيل المستندات  
- Memory agent — وكيل الذاكرة  
- PMO agent — وكيل PMO  

### AI feature highlights
- Auto scheduling from BOQ  
- Risk detection + Monte Carlo simulation  
- Status report generator  
- Resource optimization  
- Gantt generator  
- Strategy generator  
- PMO & C-level analysis  
- Management plan generation  
- Meeting summaries / AI notetaker  
- Voice messages to AI (mobile)  
- Interactive AI replies (charts, info cards)  
- Blueprint item extraction  

### Sample AI recommendation style
Actionable, severity-tagged, tied to schedule/resources (see Risks section).
