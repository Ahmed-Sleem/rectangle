# AI Construction Specialists

These are **not** full Tornix replacements — they are **intelligence wedges** a platform should integrate or partially absorb.

---

## Buildots — Construction intelligence / progress

| | |
|--|--|
| **Site** | https://buildots.com |
| **Core idea** | 360° capture (often hardhat cameras) + AI vs BIM/schedule → know what’s built |
| **Loop** | Know sooner → Act faster → Outperform |
| **Buyers** | Large GCs / owners (JE Dunn, Intel, Mace… logos) |
| **GUI** | Progress dashboards, location-based status, risk/action workflows |
| **Backend** | CV pipeline, BIM alignment, schedule comparison, heavy storage |
| **Pricing** | Enterprise / per area cited ~$3–5/sqft in some blogs (SECONDARY) |
| **Integrations** | Feeds PM platforms |

**Lesson:** Ground-truth progress beats manual % complete. Partner before build.

---

## ALICE Technologies — Generative scheduling

| | |
|--|--|
| **Site** | https://www.alicetechnologies.com |
| **Core idea** | Simulate millions of construction sequences; optimize duration/cost/resources |
| **Products** | ALICE Plan, Optimize, Model |
| **Inputs** | P6/MSP schedules + drawings (pdf/jpg/png/svg) |
| **GUI** | 2D canvas on drawings, Gantt overlay, timelapse sequence, what-if |
| **Claims** | $127B+ projects deployed; schedule compression case studies |
| **Price** | Enterprise often cited $50k–$150k/yr (SECONDARY) |

**Lesson:** BOQ/schedule AI is a category. Tornix “auto schedule from BOQ” competes narratively with ALICE+Togal+estimators — be honest about depth.

## How specialists map to Tornix AI agents

| Tornix agent idea | Specialist analog |
|-------------------|-------------------|
| Scheduling agent | ALICE + nPlan + P6 |
| Cost agent | Unifier / Procore financials + EVM |
| Risk agent | nPlan + Construction IQ |
| Documents agent | Trunk Tools + Assist |
| Reporting agent | All suites |
| Memory / playbooks | Weakly served — opportunity |
| Progress truth | Buildots / OpenSpace |

**Platform strategy:** be the **agent orchestration + system of engagement**; call specialists via API for CV/generative schedule rather than boiling the ocean.
