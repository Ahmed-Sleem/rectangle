# Residential & Field-First Construction Tools

## Buildertrend
| | |
|--|--|
| **Best for** | Residential builders, remodelers, 2–50 person firms |
| **Strengths** | Client portal, selections, change orders, scheduling, job costing, progress photos for homeowners |
| **GUI** | Client-friendly, consumer-grade polish vs enterprise AEC tools |
| **Weak** | Not commercial mega-project / deep BIM / P6 |
| **Price** | Subscription tiers; figures ~from $99/mo appear in roundups (SECONDARY, verify) |

## CoConstruct (often compared / related market)
- Custom home + client communication  
- Selections, change orders, pay apps, QuickBooks  
- Owner-facing progress  

## Fieldwire (Hilti)
| | |
|--|--|
| **Best for** | On-site coordination |
| **Strengths** | Mobile-first plans, tasks on drawings, punch, RFIs, offline, messaging |
| **Price** | Free tier; paid ~$39–$54/user/mo cited |
| **Weak** | Not full financial OS; pair with office system |
| **UX gold** | Fast sheet navigation, task pins, trade-friendly |

## Raken
- Daily reports, time, photos — field reporting specialist  
- Lower price point ~$15/user/mo cited  

## PlanGrid (legacy)
- Absorbed into Autodesk Build — established offline drawings + punch patterns everyone copies  

## Contractor Foreman / JobTread
- SMB all-in-one: estimates, invoices, PM  
- Affordable Procore alternatives  

## Bluebeam Revu
- PDF markup / takeoff standard — often **beside** PM tools, not replaced  

## Planera
- Visual CPM for contractors leaving P6/MSP complexity  
- P6/MSP import-export; Procore/ACC integrations  
- Pull planning boards; iPad field schedule  

## UX patterns to steal
1. **Sheet → pin → task** (Fieldwire/PlanGrid)  
2. **Homeowner/client portal** simplicity (Buildertrend) — analog: investor portal in Tornix demo  
3. **Daily report in <2 minutes** (Raken)  
4. **Offline-first** assumption for sites  

## Backend notes
Mostly multi-tenant SaaS; mobile offline sync queues; image pipelines. Hilti ownership gives Fieldwire distribution into tool channels.
