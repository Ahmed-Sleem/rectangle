## Primavera P6 EPPM — VERIFIED themes

### Capabilities
- Plan/schedule/control large programs and single projects  
- Multi-user schedule access; open/schedule multiple projects  
- CPM engine (the reference implementation mentally for the industry)  
- Resource & role optimization, demand/capacity, what-if  
- Team member status interfaces  
- Adaptable views  
- Integrated cost/schedule stories **via Unifier linkage** (CBS/WBS, cash flow curves, change impact, forecasts, activity sheets fed from P6)

### GUI / UX
- **Power-user scheduler UI**: tables + Gantt, dense columns, activity codes, filters  
- Not “consumer SaaS pretty” — trained planners live here  
- Web EPPM improved over client-server era but still specialist  
- Executives rarely live in P6 — need dashboards elsewhere (Tornix/Procore/Power BI)

### Backend signals
- Enterprise Oracle stack; scalable to **millions of activities** (marketing)  
- On-prem and cloud deployment histories  
- XER/XML exchange = industry interchange format everyone must speak  
- Migration program pushing customers **P6 → Primavera Cloud**

## Oracle Primavera Cloud
- Combines **CPM contract scheduling + task management** in one cloud  
- Resource + risk management  
- Strategic Oracle bet vs aging P6 UX  

## Aconex
- Document control king on large GCC/infra jobs (SECONDARY regional reviews)  
- Correspondence, transmittals, strict audit trails  
- Owner/contractor common data environment  
- Weak as full AI PMO; strong as compliance docs spine  

## Strengths
- Scheduling credibility no startup has on day one  
- Portfolio scale  
- Owner controls (Unifier)  
- GCC mega-project presence (Aconex + P6)

## Weaknesses
- Not a friendly field collab app  
- Implementation heavy  
- AI narrative exists but product DNA is controls, not chat  
- UX barrier for site engineers / Arabic first users  

## Mandatory interop for Tornix-like
**Must import/export P6** (Tornix already claims this). Treat P6 as system-of-record for master schedule on mega jobs; be the **execution + AI + Arabic layer** around it.
