# MENA / GCC Regional Competitors & Fit

Tornix’s “صفر منافسين بالعربية” is **marketing**. Regional and localized options exist; depth varies.

## Banamind
- Positioned for **GCC** contractors  
- WhatsApp-native site communication  
- Arabic language support  
- Scheduling, docs, RFI, progress, cost reporting  
- Mid-market commercial/infra/mixed-use  
- Explicitly contrasted with Procore complexity  

## FlowTrakker
- UAE/GCC purpose-built claims  
- Municipality submission formats (Dubai/Abu Dhabi DM)  
- AED contracts, VAT, retention  
- Multi-tier subcontractor chains  
- Bilingual UI  
- Mid-tier AED 10M–200M projects narrative  

## Gear Up (UAE)
- Construction PM + asset tracker  
- Kanban, Gantt, budget, WPS payroll integration  
- FTA-compliant invoicing  
- Fast implementation claims (~14 days)

## CMCS / PMWeb
- Enterprise owner/infra standard in GCC government narratives  
- Bilingual AR/EN  
- High-end licensing  

## Cezcon, Axolon ERP, First Bit, Penieltech, CloudMe
- Regional ERP/PM blends with Arabic  
- Varying construction depth  

## Global tools in GCC
| Tool | GCC note |
|------|----------|
| Procore | Deployed on large SA/UAE jobs; Arabic UI limited; expensive |
| Aconex | Strong on infra/owner document control |
| Primavera | Ubiquitous for mega schedules |
| Autodesk | Dubai/Riyadh presence |
| Zoho Projects | Affordable AR support for SMBs |

## Tornix / Ailigent context
- Saudi-oriented marketing examples  
- Ailigent HQ Riyadh  
- Engineering footprint Egypt  
- Full RTL product claim is the right fight  

## Regional buying criteria (synthesize)
1. Arabic RTL + mixed AR/EN crews  
2. WhatsApp (not only Slack/Teams)  
3. Local tax/retention/municipality docs  
4. Data residency sensitivity  
5. Implementation speed & Arabic training  
6. P6 familiarity among planners  
7. Owner vs contractor vs consultant roles on same project  

## Opportunity
A product that is **genuinely AI-native** (not just translated Procore) + **Arabic-first** + **P6-friendly** + **WhatsApp** still has room — especially mid-market that cannot digest Procore ACV.
